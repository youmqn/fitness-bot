# Fitness Coach Bot

Telegram-бот, который помогает достигать фитнес-цели: набор массы, снижение веса или поддержание формы.

## Что умеет
- Расчёт BMR/TDEE и нормы КБЖУ (Mifflin-St Jeor)
- AI-генерация меню на день под твои цифры
- Лог еды через AI (`/съел 200г куриной грудки и 100г риса`)
- Дневник веса + график динамики
- Напоминания (завтрак, обед, ужин, вода, взвешивание)
- AI-тренер: задавай свободные вопросы

## Команды
| Команда | Действие |
|---|---|
| `/start` | Начать / анкета |
| `/профиль` | Параметры и КБЖУ |
| `/рацион` | Норма КБЖУ |
| `/меню` | AI составит меню на день |
| `/съел <что>` | Записать приём пищи |
| `/сегодня` | Что съел сегодня + остаток |
| `/отменить` | Удалить последний приём |
| `/вес [кг]` | Записать вес |
| `/прогресс` | График веса |
| `/напомнить` | Настроить напоминание |
| `/напоминания` | Список напоминаний |
| `/спросить <вопрос>` | AI-тренер |
| `/цель` | Поменять цель |
| `/помощь` | Справка |

Свободный текст без команды → AI-тренер. Если в тексте «съел/выпил/завтрак» — попадёт в лог еды.

## Локальный запуск
```bash
pip install -r requirements.txt
cp .env.example .env
# заполни TELEGRAM_TOKEN, GROQ_API_KEY уже есть
python bot.py
```

## Деплой на Railway
1. Создай новый проект → Deploy from GitHub repo
2. ENV переменные:
   - `TELEGRAM_TOKEN` — от @BotFather
   - `GROQ_API_KEY` — Groq ключ
   - `MODE=webhook`
   - `WEBHOOK_URL=https://<твой-домен>.up.railway.app`
3. Railway сам выставит `PORT`

## Архитектура
- `bot.py` — handlers, ConversationHandler для анкет
- `db.py` — SQLite (aiosqlite)
- `nutrition.py` — формулы BMR/TDEE/макросы
- `ai.py` — Groq (parse_food, generate_menu, coach_advice)
- `charts.py` — matplotlib график веса
